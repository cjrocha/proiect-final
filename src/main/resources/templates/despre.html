                 Product Comparator

Product comparator scris in Java

 

Acesta este primul program scris de mine in Java folosind Spring Boot si a gramada de alte librarii, printer care pot enumera chrome driver, selenium, Jaunt si Jauntium.

 

Am elaborat acest program in idea sa ma ajute in munca mea de zi cu zi, in care trebuie sa verific si sa fac update la produse aflate in baza de date, precum si sa adaug produse noi.

 

Aplicatie este conceputa ca un "stand alone" program cu o interfata "user friendly" ce poate fi atasat unui site care e deja implementat, poate fi pusa pe un server si rulata de acolo sau poate fi rulata local - asa cum facem acum :) - pe un aparat ce are java instalat.

 

Comparatorul ofera user-ului mai multe optiuni:
1. Cautarea online in siteuri externe a produselor dorite pe baza unei suite de cuvinte cheie.

2. Afisarea si sortarea produselor existente in baza de date, pentru a reduce timpul de lucru.

3. Filtrarea produselor din baza de date pe baza unei suite de cuvinte cheie

4. Exportul listei filtrate in format csv.

 

Sper ca acest program sa fie imbunatit si sa devina un tool - chiar si personal :) - care sa fie folositor.
