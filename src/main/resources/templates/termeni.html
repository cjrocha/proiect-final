                 Product Comparator

Termeni - Product comparator
Java Based

 

Acest program nu este destinat vanzarii sau generarii de venituri in niciun fel. El a fost dezvoltat pe baza unor librarii gratuite disponibile online.

Acest program face parte din procedura de curs a Scolii Informale de IT si reprezinta proiectul final.

Datele achizitionate si folosite de program sunt disponibile online in mod gratuit oricarei persoane si nu sunt/vor fi destinate generarii de venituri.

Procedeele de "crawling", "scraping", "handling" folosite in acest program respecta "robots.txt" ale site-urilor target si nu aduc nicio modificare datelor initiale ale site-urilor sursa.
